# foggie-portal-mobile
An H5 project

# Technical Architecture
vue3 + vite + pinia + ts + （nutui / vant） + sass  + axios 

### To start a development server:

If you use [Yarn](https://yarnpkg.com/en/):

yarn dev

### To build:

With Yarn:

`yarn build`

or with NPM:

`npm run build`
